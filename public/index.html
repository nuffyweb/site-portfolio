Немного обо мне

Мне нравится воплощать макеты PSD в интерактивные сайты.

Использую при этом семантическую разметку HTML5, валидный CSS3 (осмысленные классы и комментарии) и БЭМ. Применяю как технологии верстки на flex, так и на float. Верстаю на чистом CSS или на Bootstrap. Адаптирую под различные экраны. Учитываю оптимальное отображение сайта во всех основных браузерах.

Реализую JavaScript (jQuery) логику (слайдеры, попапы,мобильное меню, анимации и т.д.) с помощю CSS и jQuery, при этом, могу использовать исключительно CSS. JavaScript (jQuery) знаю на уровне достаточном для подключения и редактирования плагинов, понимания чужих скриптов и возможности написания своих.

Для ускорения верстки использую Сборщик проектов Gulp(писала его сама для себя) Легко разбираюсь в чужом коде. Постоянно учусь и самосовершенствуюсь.
